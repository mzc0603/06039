# 像素拳皇格斗游戏 - 规格说明书

## 1. 项目概述

- **项目名称**: 像素拳皇 (Pixel KOF)
- **项目类型**: 2D像素风格格斗网页游戏
- **核心功能**: 经典街机格斗体验，支持人机对战和双人对战
- **目标用户**: 怀旧玩家、移动端和PC端用户

## 2. 技术实现

### 核心技术栈
- **前端**: 纯HTML5 + CSS3 + JavaScript (无依赖)
- **渲染**: Canvas 2D API
- **音效**: Web Audio API
- **存储**: LocalStorage (分数记录)
- **部署**: 单文件HTML，可直接部署

### 响应式设计
- 全屏自适应画布，保持16:9比例
- 移动端虚拟按键覆盖层
- PC端键盘控制
- 自动检测设备类型

## 3. 视觉与渲染规格

### 场景设置
- **画布尺寸**: 动态适应屏幕，保持960x540基准比例
- **背景**: 多层滚动背景（城市天际线+地面）
- **视角**: 侧面2D视角

### 角色设计（像素风格）
- **分辨率**: 角色 sprites 64x64 像素
- **动画帧率**: 10-15 FPS
- **角色数量**: 4个可选择角色
  - 龙 (DRAGON) - 均衡型
  - 虎 (TIGER) - 速度型
  - 熊 (BEAR) - 力量型
  - 鹤 (CRANE) - 技巧型

### 像素艺术风格
- 使用 `image-rendering: pixelated` 保持像素锐利
- 限制调色板数量（每角色8-16色）
- 复古CRT扫描线效果（可选）

### UI设计
- **血条**: 顶部横条，红色渐变，带闪烁效果
- **计时器**: 中央顶部，99秒倒计时
- **角色头像**: 血条下方，32x32像素
- **必杀技槽**: 血条下方，蓄力式
- **得分**: 右上角
- **字体**: 像素风格，使用位图字体

## 4. 游戏系统规格

### 战斗系统
- **基础攻击**: 轻拳/轻腿（快速，伤害5-10）
- **重攻击**: 重拳/重腿（慢速，伤害15-25）
- **必杀技**: 消耗必杀槽，造成30-50伤害
- **防御**: 减少50%伤害
- **投掷**: 靠近时特定键触发
- **连击系统**: 最多5连击，伤害递减

### 角色属性
| 角色 | 生命值 | 速度 | 攻击力 | 必杀技伤害 |
|------|--------|------|--------|------------|
| 龙   | 100    | 5    | 10     | 40         |
| 虎   | 80     | 8    | 8      | 35         |
| 熊   | 130    | 3    | 15     | 50         |
| 鹤   | 90     | 6    | 12     | 45         |

### 物理系统
- **重力**: 0.8 px/frame²
- **跳跃力**: -15 px/frame
- **移动速度**: 5 px/frame
- **碰撞盒**: 矩形检测
- **攻击判定**: 帧精确检测

### AI系统
- **难度等级**: 简单/普通/困难
- **行为模式**:
  - 追击距离过远的玩家
  - 随机攻击间隔
  - 防御玩家攻击
  - 偶尔使用必杀技
  - 困难模式下反应更敏捷

## 5. 音效系统

### 音效类型
- **攻击音效**: 拳击/踢腿
- **命中音效**: 打击声
- **必杀技**: 特殊音效
- **胜利/失败**: 结局音乐
- **背景音乐**: 战斗BGM（8-bit风格）

### 实现方式
- Web Audio API 合成音效
- 可静音控制

## 6. 控制方案

### PC键盘
| 按键 | 动作 |
|------|------|
| A/D | 左右移动 |
| W | 跳跃 |
| S | 下蹲/防御 |
| J | 轻拳 |
| K | 重拳 |
| L | 轻腿 |
| ; | 重腿 |
| U | 必杀技 |
| Enter | 开始/确认 |

### 移动端虚拟按键
- **左侧**: 方向键（上下左右）
- **右侧**: 动作键（拳/腿/必杀）
- 大按钮设计，适合触摸
- 实时显示按键反馈

## 7. 游戏流程

### 状态机
```
START -> SELECT -> FIGHT -> ROUND_END -> GAME_OVER -> START
```

### 回合系统
- 先赢2局者获胜
- 每局99秒
- 时间结束生命值高者胜
- KO判定：生命值≤0
