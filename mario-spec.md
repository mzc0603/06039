# 超级马里奥网页闯关游戏 - 规格说明书

## 1. 项目概述

- **项目名称**: 超级马里奥 HTML5 (Super Mario Web)
- **项目类型**: 2D横版平台闯关网页游戏
- **核心功能**: 经典马里奥游戏体验，原版关卡设计
- **目标用户**: 怀旧玩家、移动端和PC端用户

## 2. 技术实现

### 核心技术栈
- **前端**: 纯HTML5 + CSS3 + JavaScript (无依赖)
- **渲染**: Canvas 2D API
- **音效**: Web Audio API 合成音效
- **部署**: 单文件HTML，可直接部署到GitHub Pages

### 画布规格
- **基准分辨率**: 256x240 像素 (NES原始分辨率)
- **缩放**: 3倍缩放至 768x720 显示
- **保持比例**: 黑色边框填充

## 3. 游戏元素规格

### 马里奥状态
| 状态 | 描述 |
|------|------|
| 小马里奥 | 初始状态，碰到敌人死亡 |
| 大马里奥 | 吃到蘑菇后可顶碎砖块 |
| 火焰马里奥 | 吃火焰花后可发射火球 |

### 地图元素
| 元素 | 描述 |
|------|------|
| 砖块 | 普通棕色砖块，可被大马里奥顶碎 |
| 问号块 | 包含道具，顶部碰撞触发 |
| 管道 | 装饰/障碍物，不可通过 |
| 地面 | 底部平台 |
| 隐形块 | 隐藏的砖块 |
| 旗杆 | 关卡终点 |

### 道具系统
| 道具 | 效果 |
|------|------|
| 蘑菇 | 变成大马里奥 |
| 火焰花 | 变成火焰马里奥 |
| 1UP蘑菇 | 增加一条命 |
| 金币 | 增加分数 |

### 敌人系统
| 敌人 | 行为 |
|------|------|
| 库巴(Goomba) | 行走，踩踏可消灭 |
| 乌龟(Koopa) | 行走，被踩变壳，壳可消灭其他敌人 |
| 栗子怪(Piranha) | 从管道中出现 |

## 4. 关卡设计

### 经典1-1关卡布局
- 起点城堡
- 教程区域（跳第一个坑）
- 第一个问号块（出金币）
- 砖块区域
- 第一个管道
- 敌人分布区
- 旗杆终点

### 关卡长度: 2144 像素

## 5. 物理系统

| 参数 | 值 |
|------|-----|
| 重力 | 0.5 px/frame² |
| 行走速度 | 3 px/frame |
| 跳跃速度 | -12 px/frame |
| 空中速度 | 2.5 px/frame |
| 摩擦力 | 0.2 |

## 6. 控制方案

### PC键盘
| 按键 | 动作 |
|------|------|
| ← → | 左右移动 |
| ↑ 或 空格 | 跳跃 |
| ↓ | 下蹲（大马里奥） |

### 移动端虚拟按键
- **左侧**: 方向键（左右）
- **右侧**: 跳跃按钮
- 大按钮设计，适合触摸

## 7. 音效系统

| 音效 | 触发 |
|------|------|
| 跳跃 | 按跳跃键 |
| 顶砖块 | 头顶碰撞砖块 |
| 吃金币 | 接触金币 |
| 吃道具 | 接触蘑菇/花 |
| 踩敌人 | 踩到敌人头顶 |
| 死亡 | 被敌人碰到 |
| 过关 | 碰到旗杆 |
| 1UP | 吃到1UP蘑菇 |

## 8. UI设计

- **分数**: 左上角
- **金币数**: 左上角金币图标+数字
- **命数**: 右上角马里奥头像+数字
- **关卡时间**: 顶部中央 (300秒倒计时)

## 9. 游戏流程

```
开始界面 → 游戏加载 → 游戏中 → 过关/死亡 → 重新开始
```

### 死亡处理
- 从起点重新开始
- 生命减1
- 生命为0时游戏结束

### 过关处理
- 播放过关动画
- 显示得分
- 进入下一关或胜利画面

## 10. 验收标准

- [ ] 马里奥可正常移动和跳跃
- [ ] 顶砖块可获得道具
- [ ] 踩敌人可消灭
- [ ] 金币和分数正确累计
- [ ] 碰到敌人会死亡
- [ ] 旗杆可触发过关
- [ ] 移动端触控流畅
- [ ] PC键盘操作正常
- [ ] 音效正常播放
